# 한국전쟁

## 프로젝트 소개

**한국전쟁**은 한국전쟁을 배경으로 한 플래시게임을 모바일 버전으로 재구성한 전략 게임입니다.

플레이어는 자신의 나라를 관리하며 경제, 군사, 연구 등 다양한 시스템을 운영하여 국가를 발전시키고 전쟁에 대비해야 합니다.

## 게임 시스템

### 1. 예산

- 게임 내 화폐 단위.
- **세금 징수**를 통해 예산을 모을 수 있음.
- 다른 변수(시설 개발, 연구 개발 등)를 올릴 때 사용됨.

### 2. 지지도

- 플레이어가 가장 신경 써야 하는 주요 지표.
- 게임 하단에 수치가 표시됨.
- **초기 값:** 60%
- 시설 개발로 최대 100%까지 상승 가능.
- 지지도가 **30% 이하**가 되면 국민이 반란을 일으켜 게임 오버, 진행 중인 게임이 초기화됨.

### 3. 경제력

- 시설 개발을 통해 증가.
- 경제력이 높으면 **연구 개발**의 고급 단계가 해금됨.

### 4. 군사력과 전투력

- **초기 군사력:** 1000
- **초기 전투력:** 100 (군사력의 10%)
- 전쟁 준비 단계에서 군사력을 올릴 수 있음.

### 5. 연구력

- 연구 개발 활동을 통해 증가.
- 연구력이 높으면 **전쟁 준비**의 고급 단계가 해금됨.

### 6. 인구수

- 자신의 나라 인구 수를 나타내는 변수.
- 시설 개발을 통해 증가 가능.
